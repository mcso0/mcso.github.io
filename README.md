# WeMake

nomad coder study - React Router 기반 웹 애플리케이션

## 프로젝트 소개

React Router를 사용하여 구축한 모던 웹 애플리케이션입니다.

## 기능

- 🚀 React Router 기반 라우팅
- ⚡️ 빠른 개발을 위한 HMR
- 🎨 TailwindCSS 스타일링
- 📱 반응형 네비게이션
- 🔧 TypeScript 지원

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

애플리케이션이 `http://localhost:5173`에서 실행됩니다.

## 빌드

프로덕션 빌드:

```bash
npm run build
```
